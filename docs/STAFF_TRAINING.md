## Getting Started

### Your First Login

1. **Open your web browser** (Chrome, Firefox, or Edge)
2. **Navigate to the CRM system:**
   - Ask your administrator for the URL
   - Typically: `http://[NAS-IP]:5173`
3. **Enter your login credentials:**
   - Email: Provided by your administrator
   - Password: Temporary password from administrator

4. **Change your password:**
   - You'll be prompted on first login
   - Choose a strong password you can remember
   - Must include:
     - At least 12 characters
     - Uppercase letters (A-Z)
     - Lowercase letters (a-z)
     - Numbers (0-9)
     - Special characters (!@#$%^&*)

### What You Can Do

As a staff member, you can:
- Look up client information
- Check in members manually
- View membership status
- Update client contact information
- View check-in history

**What You Cannot Do:**
- Create new staff users
- Delete clients
- Access system settings
- Reset passwords (contact admin if you forget yours)

### Logging Out

**Always log out when done!**

1. Click your name in the top-right corner
2. Click **Logout**

**Or:** Simply close the browser tab (you'll be logged out automatically after 30 minutes of inactivity)

## Looking Up Clients

### Using the Search Bar

1. **Go to Clients page**
2. **Type in the search bar:**
   - Client name (first or last)
   - Phone number
   - Email address

3. **Results appear as you type**

### Search Examples

**By Name:**
- Type: "John" → Shows all clients named John
- Type: "Smith" → Shows all clients with last name Smith

**By Phone:**
- Type: "555-1234" → Shows clients with that phone number
- Type: "5551234567" → Exact phone match

**By Email:**
- Type: "john@example.com" → Shows client with that email

### Viewing Client Details

1. **Find client using search**
2. **Click on their name**
3. **View full profile:**
   - Contact information
   - Membership status
   - Check-in history
   - Notes

## Checking In Members

### Kiosk Check-In (Normal Process)

**Members should use the kiosk:**
1. Direct them to the kiosk station
2. They enter their phone number (10 digits)
3. They select their name
4. They click "Check In"
5. System records the check-in automatically

**Your role:** Help if they have trouble

### Manual Check-In

**When to use manual check-in:**
- Kiosk is not working
- Member forgot their phone
- New member needs assistance
- Special circumstances

**Steps:**
1. **Navigate to Check-ins page**
2. **Click "Manual Check-In" button**
3. **Search for the client:**
   - Type name or phone number
   - Select correct client from results
4. **Click "Check In" button**
5. **Confirm success message**

### Verifying Check-In

**After check-in:**
- Success message appears
- Client's name shows in recent check-ins
- Time stamp shows current time

**If it doesn't work:**
- Check if client exists in system
- Verify membership is active
- See [Troubleshooting](#troubleshooting) section

## Managing Client Information

### Viewing Client Information

1. **Search for client**
2. **Open their profile**
3. **View sections:**
   - Basic Info (name, phone, email)
   - Membership Status
   - Contact Methods
   - Check-In History
   - Notes

### Updating Contact Information

**When to update:**
- Client changes phone number
- Client changes email
- Client moves (address change)
- Information is incorrect

**How to update:**
1. **Open client profile**
2. **Click "Edit" button**
3. **Update fields:**
   - Phone: Must be 10 digits
   - Email: Must be valid format
   - Other information as needed
4. **Click "Save Changes"**

**Important:**
- Phone number changes affect kiosk check-in
- Tell client their new phone number for check-in

### Adding Notes

**Use notes for:**
- Special requests
- Issues or incidents
- Member preferences
- Important information

**Adding a note:**
1. **Open client profile**
2. **Scroll to "Notes" section**
3. **Click "Add Note"**
4. **Type your note:**
   - Be clear and professional
   - Include date/time if relevant
   - Include your name if not automatic
5. **Click "Save"**

**Example Notes:**
- "Prefers email contact only - 10/15/25"
- "Lost membership card, issued temporary - JD 10/15/25"
- "Regular Tuesday/Thursday visitor"
- "Birthday: March 15th"

## Viewing Memberships

### Understanding Membership Status

**Active (Green):**
- Currently valid membership
- Can check in normally
- All access granted

**Expired (Red):**
- Membership has ended
- Cannot check in via kiosk
- Needs renewal

**Pending (Yellow):**
- Membership starts in future
- Not yet active
- Cannot check in yet

### Checking Membership Status

1. **Search for client**
2. **Open their profile**
3. **View "Membership" section:**
   - Status indicator (Active/Expired/Pending)
   - Start date
   - End date
   - Days remaining

### What to Tell Members

**If Active:**
- "Your membership is active until [date]"
- "You have [X] days remaining"

**If Expiring Soon:**
- "Your membership expires on [date]"
- "Would you like to renew today?"
- Direct them to admin/manager for renewal

**If Expired:**
- "Your membership expired on [date]"
- "Let me get a manager to help you renew"
- Offer day pass if available

**Note:** You may not be able to process renewals yourself - check with your manager about your facility's process

## Common Scenarios

### Scenario 1: New Member Walk-In

**Situation:** Someone wants to join

**Your Steps:**
1. **Greet warmly:** "Welcome to BEC!"
2. **Explain membership options:**
   - Daily pass
   - Monthly membership
   - Annual membership
3. **Get manager/admin** to process signup
   - They'll create the account
   - They'll set up membership
   - They'll process payment
4. **Show them the kiosk:**
   - Demonstrate how to check in
   - Have them try it
   - Answer questions

### Scenario 2: Member Can't Check In at Kiosk

**Situation:** Member says kiosk isn't working or can't find their name

**Your Steps:**
1. **Stay calm and helpful:** "No problem, I can help with that!"
2. **Get their information:**
   - Ask for phone number
   - Or ask for name
3. **Look them up in system:**
   - Search in Clients page
   - Check if they exist
4. **Check membership status:**
   - Is it active?
   - Has it expired?
5. **Solutions:**
   - **If active:** Do manual check-in
   - **If expired:** Get manager for renewal
   - **If phone wrong:** Update phone number, show new number
   - **If not found:** Get manager to create account

### Scenario 3: Member Forgot Phone Number

**Situation:** Member can't remember which phone they used

**Your Steps:**
1. **Ask for their name**
2. **Search by name in system**
3. **Find their profile**
4. **Tell them the phone number:** "I see you're registered with [phone number]"
5. **Write it down for them** (or let them take a photo)
6. **Do manual check-in** for today
7. **Suggest:** "Save this number in your phone for next time"

### Scenario 4: System is Slow or Not Responding

**Situation:** Computer/system is frozen or very slow

**Your Steps:**
1. **Don't panic - members are waiting**
2. **Use manual check-in method** if possible
3. **If completely frozen:**
   - Write down names on paper temporarily
   - Let members in (they're here to play!)
   - Enter check-ins manually when system is back
4. **Contact administrator:**
   - Via phone/text
   - Explain the issue
   - Let them know members are being tracked on paper
5. **Communicate with members:**
   - "We're having a temporary technical issue"
   - "We've got you checked in"
   - "Sorry for the inconvenience"

### Scenario 5: Member Says Membership Should Be Active

**Situation:** Member says they paid/renewed but system shows expired

**Your Steps:**
1. **Check system carefully:**
   - Look at membership dates
   - Check notes for any info
2. **Check with manager:**
   - Maybe payment is processing
   - Maybe renewal wasn't entered yet
3. **While waiting:**
   - Do manual check-in
   - Let them use facilities
   - Be courteous and apologetic
4. **Get manager to verify:**
   - Check payment records
   - Update system if needed
   - Communicate resolution to member

### Scenario 6: Client Information Is Wrong

**Situation:** Member says name is spelled wrong, or other info is incorrect

**Your Steps:**
1. **Apologize:** "Sorry about that, let me fix it right now"
2. **Open their profile**
3. **Click Edit**
4. **Ask them for correct information:**
   - Spell name carefully
   - Verify phone number
   - Confirm email
5. **Update and save**
6. **Verify with them:** "Is this correct now?"
7. **If phone changed:** "Use this new number for check-in from now on"

## Troubleshooting

### Can't Find a Client

**Possible reasons:**
- Name is spelled differently than you think
- They're not in the system yet
- Using different phone number
- Using nickname vs real name

**Try:**
1. Search different spelling
2. Search by phone number
3. Search by email
4. Ask them how their name is spelled
5. Ask manager if client should exist

### Check-In Button Not Working

**Try:**
1. Refresh the page (F5 or Ctrl+R)
2. Click back and try again
3. Try different browser
4. Contact administrator if continues

### Can't Edit Client Information

**Possible reasons:**
- You don't have permission (some facilities restrict this)
- System error

**Try:**
1. Refresh and try again
2. Ask administrator/manager
3. They may need to make the change

### Forgot Your Password

**You cannot reset it yourself!**

**Steps:**
1. Contact your administrator or manager
2. They will reset it for you
3. They'll give you temporary password
4. Login and change to new password

## Best Practices

### Customer Service

**Be Friendly:**
- Greet members warmly
- Smile and make eye contact
- Use member's name if you know it

**Be Helpful:**
- Answer questions patiently
- Show members how to use kiosk
- Assist with any issues promptly

**Be Professional:**
- Stay calm if there's a problem
- Don't blame the system or members
- Find solutions, not excuses

### Using the System

**Search Efficiently:**
- Use full name when possible
- Use phone number if name unclear
- Double-check spelling

**Keep Information Updated:**
- Fix errors when you find them
- Add notes for important information
- Tell manager about system issues

**Protect Privacy:**
- Don't share member information
- Log out when you leave desk
- Don't leave screen visible to others

### End of Shift

**Before you leave:**
- [ ] Check for any pending manual check-ins
- [ ] Add any notes you need to
- [ ] Log out of system
- [ ] Report any issues to next shift or manager
- [ ] Lock computer if in shared space
